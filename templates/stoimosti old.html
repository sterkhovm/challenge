Форматы курсов и стоимость
Стандартный курс Занятия в мини-группах. Группы делятся по уровню знаний ученика: базовые, средние, продвинутые. Для определения уровня ученик пишет Вступительную работу. Стоимость занятий зависит от количества изучаемых предметов.
Длительность курса:16 / 32 недели
Скачать прайс PDF 1.4 Мбайт
Индивидуальный курс Вы занимайтесь индивидуально с одним из наших преподавателем по персональной программе, составленной с учетом уровня, целей и способностей ученика. Методист курирует процесс обучения.
Длительность курса:от 4 недель
Скачать прайс PDF 1.4 Мбайт
Интенсивный курс Когда времени осталось немного — необходимо повысить интенсивность. Занятия проходят в группах до 10 человек, длятся 4.5 ак. часа (180 минут с перерывом). Проработанная и испытанная программа позволяет достичь максимального КПД за минимальное время.
Длительность курса:2-6 занятий
Скачать прайс PDF 1.4 Мбайт
Спецкурс Есть ряд тем, которые, во-первых, достаточно сложны, во-вторых, требуют дополнительного времени для полноценного объяснения. К примеру, задачи с параметром в ЕГЭ по математике, либо подготовка к итоговому сочинению по литературе.
Длительность курса:2-6 занятий
